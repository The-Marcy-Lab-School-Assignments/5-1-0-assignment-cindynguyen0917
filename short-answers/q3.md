# Question
In a code snippet, demonstrate a `User` class with at least one:
- class property
- class method
- instance property
- instance method

# Answer
```class User {
    static #followers: []
    constructor (name, age) {
        this.name = name;
        this.age = age;
    }
    addFollower(username) {
        User.#followers.push(username)
        return username
    }
   static getFollowers() {
    return [...User#followers]
   }
}
```
